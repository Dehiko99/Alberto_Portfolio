# [Project 1: Books Recommendation System](https://github.com/Dehiko99/Amazon-Recommendation-System)

- Neural Collaborative Filtering (NCF) recommendation system for explicit rating prediction. The model learns user-item interaction patterns through deep neural networks to estimate user preferences. 
- Implemented a deep learning architecture that learns latent user and item embeddings to capture complex interaction patterns
- Trained and evaluated the model using explicit feedback, optimizing its ability to generate personalized recommendations.

# 🎬 [Project 2: YouTube Comments Analysis — Sentiment & Topic Modeling](https://github.com/Dehiko99/RoBERTa-Youtube-Reviews)

- Designed an end-to-end NLP system for large-scale analysis of YouTube movie comments.
- Scraped comments without relying on the YouTube API, using Python requests for data collection.
- Integrated a database layer to manage raw and processed data throughout the pipeline.
- Combined RoBERTa sentiment analysis with BERTopic topic modeling to uncover audience opinions and discussion themes.
- Automated the complete workflow from data acquisition to insight generation using Python, Hugging Face Transformers, BERTopic, and SQL.
